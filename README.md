# Flask OpenAI Chat Application

A Flask-based web application that uses the OpenAI API to process user prompts and provide AI-generated responses.

## Features

- Accepts user prompts via a POST request.
- Processes prompts using OpenAI's GPT-3.5-turbo model.
- Optionally handles uploaded files (future implementation scope).
- Logs requests and responses for debugging purposes.

## Requirements

- Python 3.7 or higher
- OpenAI API key (stored in a `.env` file)
